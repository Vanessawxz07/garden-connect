# TradeKitsune 背景信息

> **Slogan:** Your trusted Roblox games trading hub.
> Trade smarter with a secure marketplace for Grow a Garden, Blox Fruits, and expanding to more Roblox games.

---

## 一、平台介绍

TradeKitsune 是一个面向欧美 Roblox 玩家的多游戏道具交易撮合平台。平台以安全、高效的交易体验为核心，围绕"帮助玩家交易得更聪明"这一使命，提供道具交易发布与匹配、价值评估工具、道具百科、市场趋势追踪、攻略内容、社区活动等一站式服务。

平台最初服务于 **Grow a Garden (GAG)** 玩家群体，目前已横向扩展支持 **Blox Fruits (BF)**，并持续拓展更多 Roblox 热门游戏。架构上采用多游戏可配置模式，公共模块（用户中心、聊天、Giveaway）跨游戏复用，各游戏拥有独立首页和功能页面。

**核心价值主张：**
- **Safe & Fast** — 验证合作伙伴，即时连接交易对手
- **Fair Deals** — WFL 计算器帮助判断交易是否公平
- **Live Values** — 每日更新的市场价值趋势
- **Learn & Grow** — 攻略、新闻、榜单助力玩家成长

## 二、页面简介

### 2.1 游戏首页 (Game Homepage)

| 维度 | 说明 |
|------|------|
| **URL** | `/{gameKey}` (如 `/growagarden`、`/bloxfruits`) |
| **主要功能** | 作为单款游戏的门户入口，聚合该游戏的核心功能模块 |
| **核心模块** | Hero区（统计数据 + CTA）、热门道具推荐、交易列表预览、价值趋势摘要、文章推荐、FAQ |
| **用户价值** | 为特定游戏玩家提供一站式入口，快速触达交易、价值查询、攻略等高频功能 |
| **适合场景** | 玩家搜索某款游戏的交易平台、从公共首页选择游戏进入、直接访问游戏品牌词 |
| **SEO关键词** | `{game name} trading`, `{game name} trade`, `{game name} marketplace`, `roblox {game name} trade site`, `{game name} item values`, `best {game name} trades` |

## 三、公共页面

除上述各游戏独立页面外，平台还包含以下跨游戏的公共页面：

| 页面 | 说明 |
|------|------|
| **公共首页** (`/`) | 多游戏入口聚合页 + 平台品牌介绍，覆盖 What We Offer、Why Choose TradeKitsune、FAQ 等模块 |
| **聊天页** (`/chat`) | 交易双方实时沟通，支持按游戏下拉筛选，含交易请求卡片和安全提醒 |
| **用户中心** (`/user/{uid}`) | 个人资料、交易记录、收藏、关注、道具图鉴，按游戏维度分类查看 |
| **Giveaway列表** (`/giveaways`) | VIP用户发起的抽奖活动聚合页，支持按游戏筛选 |
| **文章系统** (`/{gameKey}/articles`) | 攻略(Guides)、新闻(News)、榜单(Lists & Rankings)，含道具信息卡片嵌入 |
