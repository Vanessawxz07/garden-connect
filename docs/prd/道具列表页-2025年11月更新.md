# 道具列表页 - 2025年11月更新

> 本文档记录道具列表页的现网版本需求，作为后续新游戏拓展专项的基线参考。

---

## 一、页面概述

道具列表页是展示所有游戏道具的入口页面，用户可通过分类浏览和搜索快速定位目标道具。

---

## 二、页面结构

### 2.1 页面头部

**标题区域：**
- H1标题：Items / Item Database
- 副标题说明文案

**搜索区域：**
- 全局搜索框
- 支持道具名称模糊搜索
- 搜索结果实时展示

### 2.2 分类导航

**分类Tab：**
- All（全部）
- Crops（作物）
- Pets（宠物）
- Eggs（蛋）
- Gear（装备）
- Tokens（代币）
- 其他分类根据游戏扩展

**交互逻辑：**
- 点击Tab切换分类
- URL同步更新（支持分享和SEO）
- 切换时保持搜索条件

### 2.3 排序控制

**排序选项：**
- 默认排序（热门优先）
- 按Value降序
- 按Value升序
- 按名称A-Z
- 按名称Z-A

**交互逻辑：**
- 下拉选择器
- 选择后立即刷新列表

### 2.4 道具列表

**列表布局：**
- 网格布局（Grid）
- 响应式适配（PC 4-5列，平板 3列，手机 2列）

**道具卡片：**
- 道具图片（正方形，居中裁剪）
- 道具名称
- 道具Value值
- 道具分类标签
- 当前交易数量提示

**交互：**
- 点击卡片跳转至单道具交易列表页
- Hover显示简要信息
- 支持懒加载

### 2.5 分页/加载更多

**分页方式：**
- 底部"Load More"按钮
- 或无限滚动加载

**加载状态：**
- 骨架屏占位
- 加载指示器

---

## 三、功能说明

### 3.1 搜索功能

**搜索逻辑：**
- 输入即搜索（debounce 300ms）
- 匹配道具名称（不区分大小写）
- 支持部分匹配

**搜索结果：**
- 高亮匹配文字
- 无结果时显示空状态提示
- 建议相似道具

### 3.2 筛选功能

**Value筛选：**
- 最小Value输入
- 最大Value输入
- 支持单边筛选

**其他筛选（按需）：**
- 是否有活跃交易
- 稀有度筛选

### 3.3 数据展示

**统计信息：**
- 当前分类道具总数
- 筛选结果数量

**空状态：**
- 无道具时显示友好提示
- 引导用户调整筛选条件

## 四、SEO优化

### 4.1 TDK设置

**Title格式：**
- 全部：[Game Name] Items - Complete Item Database
- 分类：[Category Name] Items - [Game Name] Trading

**Description格式：**
- 包含道具类型、数量、交易相关关键词
- 最多160字符

**Keywords：**
- 游戏名称
- 道具分类名称
- trading、value、items等

### 4.2 URL结构

**路由设计：**
- 全部道具：/items
- 分类道具：/items?category=crops
- 搜索结果：/items?search=keyword

**URL参数：**
- category: 分类筛选
- search: 搜索关键词
- sort: 排序方式
- page: 页码

### 4.3 结构化数据

**ItemList结构化数据：**
```json
{
  "@type": "ItemList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "item": {
        "@type": "Product",
        "name": "道具名称",
        "image": "道具图片URL"
      }
    }
  ]
}
```

### 4.4 标题层级

- H1: 页面主标题（唯一）
- H2: 分类标题
- H3: 道具名称（列表项）

## 五、性能优化

### 5.1 图片优化

- 使用WebP格式
- 响应式图片（srcset）
- 懒加载（Intersection Observer）
- 占位图过渡

### 5.2 数据加载

- 首屏优先加载
- 分页数据懒加载
- 接口缓存策略

### 5.3 渲染优化

- 虚拟列表（大数据量时）
- 骨架屏占位
- 防抖搜索

## 六、响应式设计

### 6.1 断点设置

- 桌面端：≥1024px，4-5列网格
- 平板端：768px-1023px，3列网格
- 移动端：<768px，2列网格

### 6.2 移动端适配

- 搜索框全宽
- 分类Tab横向滚动
- 筛选器收起为抽屉
- 卡片简化显示
