## 一、页面结构

### 1.1 Hero区域

- 页面主标题：All Tradeable Pets & Crops In Grow A Garden
- 副标题：Explore a vibrant marketplace of pets and crops, updated weekly!
- 统计数据：Ongoing Trades / Completed Trades
- CTA按钮：POST A TRADE
- 快捷入口：Pet Value Trends、Pet Value Calculator

### 1.2 分类Tab

- All（全部）
- Pets（宠物）
- Crops（作物）

### 1.3 筛选和排序

**筛选：**
- 搜索框：支持道具名称搜索
- 稀有度筛选：All Rarity下拉

**排序：**
- Newest Added（最新添加）

### 1.4 道具列表

**统计：**
- 显示当前可交易道具数量（如：574 Items Available For Trading Now）

**卡片内容：**
- 道具图片
- 稀有度标签（如Rare、Divine、Mythical、Legendary等）
- 道具名称 + 外链图标
- X People Want（需求人数统计）
- TRADE按钮

**布局：**
- 网格布局，每行7个卡片（桌面端）
- 底部分页器

### 1.5 Pet Value List区域

**Tab切换：**
- Pets
- Huge Pets
- My Pets

**表格内容：**
- Rank（排名，带趋势箭头）
- Pet（图片+名称+稀有度标签）
- Value
- Tokens
- Daily Growth（涨跌百分比）
- Offers
- Requests

**底部：**
- VIEW MORE按钮

### 1.6 底部CTA区域

- 标题：Your Ultimate GAG Trading Hub
- 描述文案
- Start Trading Now按钮
- 统计数据：Ongoing Trades / Completed Trades

---

## 二、交互说明

- 点击道具卡片跳转至单道具交易列表页
- 点击TRADE按钮进入交易流程
- 分类Tab切换刷新列表
- 搜索框输入即搜索
